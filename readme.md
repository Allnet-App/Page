## Installation

### Composer requirement
Add package to your `require` key in the `composer.json` file:

``` json
"asgardcms/page-module": "dev-master"
```

Followed by a composer update.

### Package migrations

Run the migrations:

``` bash
php artisan module:migrate Page
```
